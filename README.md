# Blinkit Sales Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing Blinkit retail sales data. It focuses on delivering insights through visual storytelling and business-driven metrics.

![Dashboard](images/dashboard1.png)

---

## Dataset
- ~8,000+ records  
- 16 product categories  
- Multiple outlet types, sizes, and locations  

---

## Tools & Technologies
- Power BI  
- DAX (YoY Growth, KPIs, Conditional Formatting)  
- Excel / Power Query (Data Cleaning)

---

## Dashboard Pages
1. Overview (KPIs + Sales Trend)  
2. Outlet Analysis  
3. Product Intelligence  
4. Insights  
5. Recommendations  

---

## Key Features
- Interactive dashboard with slicers and navigation  
- Line + Column chart for sales trend analysis  
- YoY growth calculation using DAX  
- Conditional formatting (Green/Red indicators)  
- Top & Bottom category analysis  

---

## Key Insights
- Top categories contribute majority of sales (~60%+)  
- Outlet type and location significantly impact performance  
- Item visibility has weak correlation with sales  
- Sales trend remains stable with minor fluctuations  

---

## Conclusion
Business performance is driven more by category demand and outlet strategy than visibility.

- Advanced KPI tracking  
- Integration with real-time data  
- Predictive analytics
